1. 깃설정
git init

2. 깃허브 환경설정 아이디(유저이름)
    git config user.name 'honeybutter900'

3. 깃허브 이메일
    git config user.email 'honeybutter900@gmail.com'

4. 배포할 리포지토리 => 오리진 등록
git 원격지접속 추가 오리진 https://github.com/honeybutter900/web01.git
git remote add origin 'https://github.com/honeybutter900/web01.git'

5. 모든 파일 선택
    git add .

6. 모든 선택된 파일의 메세지 등록
    git commit -m '웹페이지 배포'

7. 모든 선택된 파일의 마스터에게 올리기
    git push 오리진(주소)에 주인에게 보낸다
    git push origin master
